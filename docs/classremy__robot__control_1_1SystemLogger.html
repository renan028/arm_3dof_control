Detailed Description

The SystemLogger is a simple logger to file csv. It handles the file creation, open, close and stream. Not generic or fast, but just for this specific scenario. The created file has the following format: 

* +------+---+---+---+----+----+----+--------+--------+--------+
* | time | x | y | z | ux | uy | uz | theta1 | theta2 | theta3 |
* +------+---+---+---+----+----+----+--------+--------+--------+
*

Do not use it for production.
